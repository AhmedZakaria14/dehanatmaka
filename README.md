# موقع دهانات مكة (dehanatmaka.com)

هذا المستودع يحتوي على نسخة كاملة ومؤرشفة من موقع **دهانات مكة**. تم تحميل الموقع وتنظيمه ليكون قابلاً للتشغيل كصفحات ثابتة (Static Pages).

## محتويات الموقع:
- **الصفحة الرئيسية**: `index.html`
- **صفحة الإيبوكسي**: `epoxy.html`
- **صفحة الدهانات الخارجية**: `exterior-paints.html`
- **صفحة الدهانات الداخلية**: `interior-paints.html`
- **الصور والوسائط**: مجلد `images/`
- **الملفات البرمجية والتنسيق**: مجلد `assets/`

## كيفية التشغيل:
يمكنك ببساطة فتح ملف `index.html` في أي متصفح ويب لعرض الموقع، أو رفعه على **GitHub Pages** ليتم استضافته مجاناً.
